# Hitos del Proyecto (Milestones)

El proyecto se divide en 3 Milestones principales asociados a cada Sprint:

### Milestone 1: Autenticación y Perfiles (Sprint 1)
* HU01 - Registro de Usuario
* HU02 - Login de Usuario
* HU03 - Registro de Mascota
* HU04 - Panel Administrativo

### Milestone 2: Gestión de Turnos (Sprint 2)
* HU05 - Visualización de Disponibilidad
* HU06 - Reserva de Turno
* HU07 - Cancelación de Turno
* HU08 - Bloqueo de Horarios por Recepción

### Milestone 3: Notificaciones e Historial (Sprint 3)
* HU09 - Recordatorio por Email automático
* HU10 - Historial Médico Básico por mascota
* HU11 - Buscador de Turnos interno
* HU12 - Alerta de Cancelación para Veterinarios
