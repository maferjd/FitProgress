# Metodología Ágil Elegida: Scrum

Para el desarrollo del proyecto "PetSalud" se ha seleccionado la metodología ágil **Scrum** de forma pura.

## Justificación de la elección
* **Timeboxing (Bloques de tiempo):** Dado que el proyecto cuenta con un límite estricto de 6 semanas para su entrega, la estructura de Scrum nos permite dividir el tiempo en **3 Sprints de 2 semanas cada uno**. Esto asegura que el equipo mantenga un ritmo constante y predecible.
* **Roles definidos:** Nos organizaremos asignando los roles clave (Scrum Master, Product Owner y Dev Team) entre los integrantes del grupo para garantizar una comunicación eficiente y una correcta gestión del backlog.
* **Eventos de Scrum:** Realizaremos las ceremonias de forma interna: Sprint Planning, Daily Scrum, Sprint Review y Retrospective.
