# Análisis de Requerimientos (Historias de Usuario)

A continuación se listan las principales Historias de Usuario para el sistema:

* **HU01:** Como dueño de mascota, quiero registrarme con mi email y contraseña para acceder al sistema.
* **HU02:** Como usuario registrado, quiero iniciar sesión de forma segura para gestionar mis turnos.
* **HU03:** Como cliente, quiero agregar el perfil de mi mascota para asociarla a mis consultas.
* **HU05:** Como cliente, quiero ver los horarios disponibles para elegir el que me convenga.
* **HU06:** Como cliente, quiero confirmar un turno para asegurar la atención médica de mi mascota.
* **HU07:** Como cliente, quiero cancelar un turno desde mi perfil para liberar el espacio si surge un imprevisto.
